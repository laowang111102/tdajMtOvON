# 酒店管理系统 java1126

## 项目概述

酒店管理系统是一款基于前后端分离模式开发的应用，主要用于实现酒店房间管理、用户预订、订单处理等功能。本项目采用Java语言进行开发，前端使用Vue.js框架和Element-UI组件库，后端采用Spring Boot框架和MyBatis Plus持久层框架。

## 技术栈

- 前端技术：Vue.js、Element-UI
- 后端技术：Spring Boot、MyBatis Plus
- 数据库：MySQL

## 功能模块

### 系统角色

- 管理员
- 用户

#### 管理员功能

- 登录
- 用户管理
- 订单管理
  - 处理酒店预约订单
  - 查看订单列表
- 房间管理
  - 房间列表
  - 详情查询
  - 添加房间
  - 修改房间
- 评论列表

#### 用户功能

- 注册
- 登录
- 查询空余房间
- 房间预约
- 我的
  - 查看预约订单
